# 今天你要喝什麼？
### 飲料檢索網站
### [DEMO 頁面](https://minnnn7716.github.io/whatToDrink/#/)
目前以五家飲料店為主，使用者可透過關鍵字搜尋飲品、瀏覽店家菜單、飲品資訊，在茫茫飲料海中篩選你的喜好飲品。

## 網站功能
* 關鍵字搜尋
* 各家菜單、飲料資料
* 飲料評等與評論
* 飲料喜愛清單
* 隨機店家、飲料抽籤

## 儲存庫分支
* main - 主要原始程式碼
* gh-pasge - 編譯建構後程式碼

## 專案指令
### 安裝相關專案
```sh
npm install
```
### 運行環境

```sh
npm run dev
```
### 建構專案
```sh
npm run build
```
